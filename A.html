Etiqueta a

Descripcion

Con esta etiqueta podemos crear un enlace o hipervinculo hacia algun link de internet, un archivo o a una seccion dentro de la propia pagina WEB para esto debemos agregar un parametro más a la etiqueta el cual es href en donde colocaremos el link, nombre del archivo, el nombre de la seccion a donde queremos que nos mande entre comillas y para el caso de las secciones debemos agregar el simbolo # antes del nombre de la seccion y de la misma manera debemos agregar en la seccion un id con el cual la vamos a reconocer para poder crear el hipervinculo más adelante.

  1. Ejemplo para crear hipervinculo a un link

    • <a href="link"> Texto que sera el hipervinculo </a>

  2. Ejemplo para crear hipervinculo a un archivo dentro de la carpeta de trabajo

    • <a href="Nombre.Extension"> Texto que sera el hipervinculo </a>

  3. Ejemplo para crear hipervinculo a un archivo de una carpeta externa

      4. Si la carpeta esta un nivel arriba de la carpeta de trabajo, es decir, es una carpeta que se encuentra dentro de la carpeta de trabajo, solamente tenemos que colocar el nombre de la carpeta seguido de / con el nombre del archivo o la ruta correspondiente para el archivo que necesitamos.

    • <a href="Carpeta/Nombre.Extension"> Texto que sera el hipervinculo </a>

      • Si la carpeta esta un nivel abajo de la carpeta de trabajo, es decir, es una carpeta que se encuentra atras de la carpeta de trabajo, tenemos que colocar al inicio dos puntos (..) para retroceder una carpeta luego / seguido del nombre de la carpeta y a continuacion el nombre del archivo o la ruta correspondiente para el archivo que necesitamos.

    • <a href="../Carpeta/Nombre.Extension"> Texto que sera el hipervinculo </a>

  4. Ejemplo para crear hipervinculo a una seccion

      5. Previamente debemos agregar un id a la seccion de esta manera: <section id="Seccion1">

    • <a href="#idseccion"> Texto que sera el hipervinculo </a>
