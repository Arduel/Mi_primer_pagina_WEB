Etiqueta hgroup

Descrpcion

Esta etiqueta tiene mayor peso para el posicionamiento SEO, pues lo que hace es relacionar etiquetas h entre ellas y otorgarles casi el mismo peso, es decir, dentro de hgroup pueden estar uma etiqueta h1, h2 y h3 (o cualquier otra etiqueta h) y les otorgora el mismo peso, por asi decirlo, los navegadores sabran que esas etiquetas estan completamente relacionadas entre ellas y seran consideradas en todas las busquedas relacionadas.

La sintaxis es muy basica, unicamente se crea la etiqueta y dentro se colocan las etiquetas que se van a relacionar

<hgroup>

  • <h1> Etiqueta H1 </h1>
  • <h2> Etiqueta H2 </h2>
  • <h3> Etiqueta H2 </h3>

</hgroup>
