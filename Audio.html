Etiqueta audio

Descripcion

La etiqueta audio nos permite insertar archivos de audio en la pagina web. Para ello es necesario colocar tres atributos:

  1. src
  2. type
  3. controls

El atributo src (source) corresponde a la ruta del archivo multimedia, en este caso de audio.

  • SINTAXIS: <audio src="Nombre.Extension"></audio>

El atributo type corresponde al tipo del archivo multimedia y su extension, en este caso de audio.

  • SINTAXIS: <audio src="Nombre.Extension" type="audio/extension"></audio>

El atributo controls nos crea los controles necesarios para la reproduccion del contenido multimedia.

  • SINTAXIS: <audio src="Nombre.Extension" type="audio/extension" controls="controls'></audio>

Estos atributos son necesarios para agregar correctamente el archivo de audio a la pagina web.
