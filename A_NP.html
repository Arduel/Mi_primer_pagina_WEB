Atributo target = "_blank"

Descripcion

Como ya vimos en la seccion de Enlaces podemos crear hipervinculos o enlaces a diferentes archivos o sitios web, pero cada que ibamos al enlace se abria en la misma ventana, es decir, se cerraba el archivo que estabamos visualizando y se sobreponia el archivo del enlace, para evitar esto podemos abrir el archivo del enlace en una pagina diferente de donde estamos y asi no tener que dejar de ver el contenido que estabamos viendo.

Para esto debemos agregar un atributo extra a la etiqueta a al momento de llamarla, este agumento es target = "_blank" con este atributo hacemos que el enlace al que vayamos sea abierto en una nueva ventana y no necesariamente en la que estamos.

  • <a href="link" target="_blank"> Texto que sera el hipervinculo </a>

Esto aplica para cualquier archivo, seccion o link.
