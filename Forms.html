Formularios

Descripción

La etiqueta form nos permite crear formularios en la página web, de esta manera el usuario interactua con ella.

Es importante introducir algunos argumentos al crear un formulario:

  • id: Es el identificador del formulario.
  • name: Es el nombre del formulario.
  • method: Es la acción que hará el formulario, en este ejemplo el tipo es post.
  • action: En este atributo se incluye el archivo de backend que trabajará con la información.

Dentro de un formulario existen dos tipos de objetos

  1. Botones
  2. Casillas de información

La diferencia entre ellos es la manera en como se declaran los argumentos, para esto nos apoyamos con la etiqueta input.

Para el caso de los botones se debe designar el parámetro type como: submit

En este tipo de objeto podemos cambiar el texto del bóton colocando el texto que deesemos en el parámetro value.

Estos botones realmente no hacen nada, pero podemos crear un bóton que nos permita regresar al inicio asignando al parámetro action el nombre del archivo de inicio.

*** Presionar el bóton te regresará a la pestaña de inicio ***

Para las casillas de información debemos asignar al parámetro type como: text

La manera de agregar un texto en la casilla es diferente ya que el tipo 'text' maneja dos argumentos diferentes a los del tipo submit

  1. placeholder: Es el texto que se muestra por defecto en la casilla
  2. onfocus: Detecta si el usuario va a escribir algo y pone el puntero dentro de la casilla, se puede combinar con Js para que cambie el texto de la casilla hasta que el usuario escriba algo
